<h2 id="como-rodar">How to run the project</h2>

```bash
    # Clone this repository
    $ git clone [https://github.com/WelderBM/mix-webapp](https://github.com/WelderBM/mix-webapp)

    # Access the project folder
    $ cd mix-webapp

    # Install the dependencies
    $ npm install

    # Run the application
    $ npm run dev
```
